## 🌸 Deltarune Chapter 5 Countdown Timer 🌸

A countdown timer widget for **Rainmeter**, dedicated to the upcoming release of **Deltarune Chapter 5**. The widget perfectly replicates the style of the original [Deltarune](https://deltarune.com/) website!

<img src="https://github.com/user-attachments/assets/552c761b-6c14-4129-9338-646bca6382fc" alt="screenshot" width="100%" />

---

**Universal Time ⌚ (UTC):** The widget automatically tracks time based on the global UTC release timestamp. It dynamically syncs with your PC's time zone to show the exact remaining days, hours, and minutes—no matter where you live!

## 💻 How to Install:
- Download and install the latest version of [Rainmeter](https://rainmeter.net) if you haven't already.
- Download the pre-packaged widget installer: [DOWNLOAD CHAPTER 5 COUNTER](https://github.com/Oscilloscope-ALT/deltarune-chapter5-countdown-timer/releases/download/v1.0.1/Deltarune.Chapter.5.timer_1.0.1.rmskin)
- Run the downloaded `Deltarune Chapter 5 timer_1.0.1.rmskin` file and click **Install** in the Rainmeter window. The widget will launch automatically on your desktop.

## 🎀 Personalization

Please note that I have not tested this widget on different screen resolutions, so some layout issues may occur. 
If you want to customize the design or optimize the widget for your monitor, open the `deltarune_chapter_5_timer` folder and adjust the parameters in the `.ini` file to fit your needs. Feel free to open an issue if you encounter any problems with the widget!
